    Angular Material
    Bootstrap
    fortawesome/angular-fontawesome

## Instalar dependencias
    ```bash
    git clone https://github.com/javieraizq88/rickAndMorty-Angular.git
    npm install
    ```

##  Para iniciar local development server:

    ```bash
    ng serve
    ```

    Se abrira en el puerto `http://localhost:4200/`
